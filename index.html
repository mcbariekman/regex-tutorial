Regex Tutorial

This tutorial will teach you about regular expressions, a powerful tool for searching and manipulating text.

Summary

The regex featured in this tutorial is [a-z]+, which matches any lowercase letter sequence.

Component 1: Character Class

A character class matches a single character from a set of characters. It's denoted by enclosing the set of characters within square brackets []. For example, the regex [aeiou] will match any lowercase vowel character.

[aeiou]

Example:

Regex: [aeiou]
String: "apple"
Match: "a"

Component 2: Quantifier

A quantifier specifies how many times a particular character or group should be matched. The most common quantifiers are + (matches one or more occurrences) and * (matches zero or more occurrences). For example, the regex [a-z]+ will match any sequence of lowercase letters.

[a-z]+

Example:

Regex: [a-z]+
String: "apple"
Match: "apple"

Component 3: Grouping

Grouping allows you to apply quantifiers and other operators to a specific section of the regex. It's denoted by enclosing the group within parentheses (). For example, the regex ([a-z]+)-\1 will match any repeated sequence of lowercase letters separated by a hyphen.

([a-z]+)-\1

Example:

Regex: ([a-z]+)-\1
String: "hello-hello"
Match: "hello-hello"
